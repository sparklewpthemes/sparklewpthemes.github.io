Install Kathmag/Kathmag Pro

There are two ways to installKathmag/Kathmag Pro theme. You can either install the theme using WordPress Theme Installer (Recommended) or you can do it via FTP.

Method One(Recommended) : Install Using the WordPress Theme Installer

  1. Log into your website's admin panel. Once logged in, you'll see your dashboard.
  2. Go to Appearance > Themes.
  3. Click on the Add New button.
  4. Click on Upload Theme button.
  5. Click on Choose File button. Locate and select the zipped theme file.
  6. Click on Install Now button.

Method Two: Install Using File Transfer Protocol (FTP)

  1. Unzip the theme file on your computer.
  2. Log into you web server using FTP Client application.
  3. Locate the wp-content folder in your WordPress installed folder.
  4. Upload the un-zipped theme file folder into the: wp-content/themes folder.

Activate the theme

  1. Once theme is installed, Click on theme.
  2. Then click on Activate button. Once done your theme will be activated.

Installing Plugins

To install a plugin in your WordPress admin panel and follow the following steps.

Plugin Installing Method

  1. Log into your WordPress site and opens the Administration Panel.
  2. Click Navigate to Plugins > Add New in the WordPress Dashboard.
  3. Search for the name of the plugin you wish to install which will return a list of potential plugins.
  4. Click Details to read more about a plugin.
  5. Click “Install Now” if you wish to install the plugin.
  6. It may ask if you are sure, and you may proceed. The installation will commence once you confirm to proceed.
  7. After the plugin is downloaded. Go to Plugins>Installed Plugins. Click “Activate Plugin” to make it available for use.

Recommended Plugins : We recommend you to download the following plugins when necessary:

  1. Jetpack by WordPress.com, by: Automatic (The plugin provides a variety of complementary features for your Theme.)
  2. Regenerate Thumnails by Alex Mills (Viper007Bond) (If you have existing content, this will help you update your image thumbnails sizes.)
  3. Woocommerce by Automattic (This plugin will help you set up your online shop. )

Enable Front Page

By default your Blog Page will be enabled but not the Front Page. Front Page should be done manually.

Enable Front Page

  1. Log into your admin panel. Then you'll see your dashboard.
  2. Go to Appearance > Customize > Front Page Options > Front Page.
  3. Check on Enable Front Page. Once done, your front page be activated.
  4. Click on Publish button to save the setting.

Note : Remember that Kathmag/Kathmag Pro theme is fully base on customizer so first configure homepage.

Configure Menu

There are three menus defined in this theme, Top Menu, Bottom Menu and Primary Menu. Top menu is at the top of the header. Bottom Menu is at the bottom of the footer, at the same row as copyright text. And Primary Menu is at the header.

  1. Go to Appearance > Menus.
  2. If there are no menu created then create one. Click on create new menu, set name of the menu, add menu items, which can be either page, post, category, custom link, etc., set the location of the menu You can also drag and drop menu items and arrange them in hierarchy. Once you are done, click on Save Menu button.

Configure Single Post

When creating post, you can set the layout of the post. There are three layouts of the post, Standard, Classic and Full Width.

  1. If you want to add new post, Go to Posts > Add New. If you want to update existing post, Go to Posts > All Posts, then select the post that you want to update.
  2. Set Title of the post.
  3. Write Content of the post.
  4. Set format of the post. If you set, format to Gallery, Video and Audio, then respective options will be enabled to set the post format content.
  5. Set category and tag for the post.
  6. Set featured image.
  7. There are three sidebar positions, Left Sidebar, Right Sidebar and No Sidebar. Select any one sidebar position.
  8. There are three layouts for the post, Standard, Classic, and Full Width. On selecting Standard layout, the featured image or video or gallery images or audio will be displayed row above the post content and sidebar. On selecting Classic layout, the media items will be displayed in same column but above of the post content. On selecting Full width, featured image will be displyed taking the width of the screen and video, audio and gallery will be displayed at in same column but above of the post.
  9. If it is new post, click on Publish button, if not click on Update button.
